## HTML
# 📄 HTML Roadmap for Beginners

---

## Phase 1: HTML Basics
### 📘 Concepts to Cover
- What is HTML & how browsers interpret it
- Structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Headings & Paragraphs
- Text formatting
- Links (`<a>`)
- Images (`<img>`)

### 🧪 Practice Tasks
- Create a “Hello World” page
- Add heading & paragraph about yourself
- Insert an image with alt text
- Create a navigation with links

---

## Phase 2: Lists, Tables, and Forms
### 📘 Concepts to Cover
- Lists (`<ul>`, `<ol>`, `<li>`)
- Tables (`<table>`, `<tr>`, `<td>`, `<th>`)
- Forms (`<form>`, `<input>`, `<textarea>`, `<select>`, `<button>`)

### 🧪 Practice Tasks
- Build a shopping list
- Create a student grade table
- Build a contact form

---

## Phase 3: Semantic HTML
### 📘 Concepts to Cover
- `<header>`, `<footer>`, `<section>`, `<article>`, `<nav>`, `<aside>`
- Accessibility basics

### 🧪 Practice Tasks
- Blog post layout
- Navigation bar
- Resume structure

---

## Phase 4: Advanced HTML
### 📘 Concepts to Cover
- HTML5 elements (`<video>`, `<audio>`, `<canvas>`)
- Embedding content (`<iframe>`)
- Metadata (`<meta>`, `<title>`, `<link>`)
- Form validation

### 🧪 Practice Tasks
- Embed a YouTube video
- Create an audio player
- Build a validated login form

---

## Phase 5: Project
### 🛠️ Mini Project: Personal Resume Page
- Profile image & About section
- Education in a table
- Contact form
- Semantic structure
