## CSS
# 🎨 CSS Roadmap for Beginners

---

## Phase 1: CSS Basics
### 📘 Concepts to Cover
- Inline, Internal, and External CSS
- Selectors, colors, fonts
- Backgrounds
- Borders, Margins, Padding (Box Model)

### 🧪 Practice Tasks
- Style an “About Me” page
- Change section backgrounds
- Style a button with hover effect

---

## Phase 2: Box Model & Positioning
### 📘 Concepts to Cover
- Display (`block`, `inline`, `flex`)
- Position (`relative`, `absolute`, `fixed`, `sticky`)
- Z-index
- Overflow

### 🧪 Practice Tasks
- Two side-by-side boxes
- Sticky navbar
- Text overlay on image

---

## Phase 3: Layouts (Flexbox & Grid)
### 📘 Concepts to Cover
- Flexbox (alignment, grow/shrink, wrapping)
- Grid (rows, columns, areas)

### 🧪 Practice Tasks
- Navbar with Flexbox
- 3-column layout
- Responsive gallery with Grid

---

## Phase 4: Responsive Design
### 📘 Concepts to Cover
- Units (`%`, `em`, `rem`, `vh`, `vw`)
- Media queries
- CSS variables

### 🧪 Practice Tasks
- Mobile-friendly About page
- Responsive navbar
- Grid layout with breakpoints

---

## Phase 5: Advanced Styling
### 📘 Concepts to Cover
- Transitions & Animations
- Pseudo-classes & Pseudo-elements
- Shadows
- Gradients
- Filters

### 🧪 Practice Tasks
- Button animation
- Bouncing ball with keyframes
- Gradient hero section

---

## Phase 6: Project
### 🛠️ Mini Project: Responsive Portfolio Website
- Header + Navbar
- About section
- Projects grid
- Contact form
- Responsive + Animations
