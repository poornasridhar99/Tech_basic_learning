## FRONTEND

# 🌐 Frontend Development Roadmap (High-Level)

---

## Phase 1: Web Foundations
- HTML Basics (document structure, text, links, images, forms, semantic tags)
- CSS Basics (selectors, colors, fonts, box model, layouts with Flexbox/Grid, responsive design)

## Phase 2: JavaScript Fundamentals
- Variables, data types, operators
- Control flow, loops, functions
- DOM manipulation & events
- Arrays, objects, ES6 features

## Phase 3: Advanced JavaScript
- Closures, scope, `this`
- Async JS (Promises, async/await, fetch API)
- LocalStorage, APIs, modules

## Phase 4: React.js (Modern Frontend)
- JSX, components, props, state
- Event handling, forms, conditional rendering
- React hooks (`useState`, `useEffect`)
- Context API, React Router
- Fetching APIs in React

## Phase 5: Frontend Projects
- Portfolio Website (HTML + CSS + JS)
- Responsive Landing Page (CSS Grid/Flexbox)
- To-Do App (React + LocalStorage)
- Weather App (React + API)
- Expense Tracker (React + Hooks)
